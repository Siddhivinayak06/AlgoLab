# Features

## Overview
AlgoLab is an interactive learning platform for understanding algorithms through visualization, metrics, and guided experimentation.

## Core Learning Features

### 1. Sorting Visualizer
- Bubble Sort, Quick Sort, and Merge Sort visualizations.
- Live bar updates for compare, swap, and sorted states.
- Step-by-step execution controls:
  - Start
  - Pause
  - Resume
  - Step Forward
  - Step Backward
  - Reset

### 2. Dataset Generator
- Random dataset generation.
- Regenerate dataset using the same configuration.
- Custom input support (comma-separated values).
- Dataset type options:
  - Random
  - Nearly Sorted
  - Reverse Sorted
  - Few Unique
  - Already Sorted
- Dataset preview with value chips and quick stats.

### 3. Learning-Oriented UI
- Collapsible beginner guide.
- Current step explanation panel.
- Color legend for visualization states.
- Sticky visualization panel to reduce scrolling friction.

### 4. Live Metrics and Progress
- Comparisons
- Swaps
- Operations
- Execution Time
- Array Size
- Progress Percentage
- Sorting progress bar

### 5. Algorithm Information
- Short explanation per algorithm.
- Complexity cards:
  - Worst Case
  - Average Case
  - Best Case
  - Space Complexity

## Additional Pages
- Dashboard
- Binary Search Visualizer
- Racing Mode
- Performance Analysis
- Experiments History
- Reports
- Admin pages (role-based)

## Experiment Tracking
- Sorting runs are saved with metadata such as algorithm, dataset type, size, speed, and runtime metrics.
- Supports review and reporting workflows.
